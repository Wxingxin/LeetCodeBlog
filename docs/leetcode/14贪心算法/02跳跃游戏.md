# 题
给你一个非负整数数组 nums ，你最初位于数组的 第一个下标 。数组中的每个元素代表你在该位置可以跳跃的最大长度。

判断你是否能够到达最后一个下标，如果可以，返回 true ；否则，返回 false 。

 

## 示例 1：

输入：nums = `[2,3,1,1,4]`
输出：true
解释：可以先跳 1 步，从下标 0 到达下标 1, 然后再从下标 1 跳 3 步到达最后一个下标。
## 示例 2：

输入：nums = `[3,2,1,0,4]`
输出：false
解释：无论怎样，总会到达下标为 3 的位置。但该下标的最大跳跃长度是 0 ， 所以永远不可能到达最后一个下标。
# 解

下面给你 **最清晰的贪心解法 + 详细例子推演**，并给出可直接使用的 JavaScript 代码。

---

# 🧠 思路（贪心法）

在遍历数组时，维护一个变量 **`maxReach`**，表示：

> **当前能够到达的最远下标位置**

每访问一个位置 `i`：

1. 如果 `i > maxReach`，说明你连当前位置都到不了 → 直接返回 `false`
2. 否则可以更新最远可达范围：

   ```
   maxReach = Math.max(maxReach, i + nums[i])
   ```
3. 如果某一瞬间 `maxReach` 已经 >= 最后一个下标，就可以返回 `true`

这是 O(n) 时间、O(1) 空间的最优解。

---

# ✅ 代码实现

```js
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canJump = function(nums) {
    let maxReach = 0; // 当前能够到达的最远下标

    for (let i = 0; i < nums.length; i++) {
        // 如果当前位置已经超过可达范围，则失败
        if (i > maxReach) return false;

        // 更新最远可达位置
        maxReach = Math.max(maxReach, i + nums[i]);

        // 已经可以到最后了
        if (maxReach >= nums.length - 1) return true;
    }
    
    return true;
};
```

# 📌 示例讲解（nums = [2, 3, 1, 1, 4]）

我们用一张表解释算法怎么跑：

| i（下标） | nums[i] | i 是否可达？  | 能跳到的最远位置 i + nums[i] | maxReach 更新后 | 说明               |
| ----- | ------- | -------- | -------------------- | ------------ | ---------------- |
| 0     | 2       | 0 ≤ 0 ✔️ | 0 + 2 = 2            | maxReach = 2 | 从 0 最远能到 2       |
| 1     | 3       | 1 ≤ 2 ✔️ | 1 + 3 = 4            | maxReach = 4 | 现在最远能到 4（最后一个下标） |
| ——    | ——      | ——       | ——                   | ——           | 已经能到最后 → 返回 true |

结果为 **true**。

# 📌 再看一个不能到达的例子

nums = `[3,2,1,0,4]`

| i | nums[i] | i 是否可达？  | 新最远 | maxReach | 说明       |
| - | ------- | -------- | --- | -------- | -------- |
| 0 | 3       | 0 ≤ 0 ✔️ | 3   | 3        | 可达       |
| 1 | 2       | 1 ≤ 3 ✔️ | 3   | 3        | 可达       |
| 2 | 1       | 2 ≤ 3 ✔️ | 3   | 3        | 可达       |
| 3 | 0       | 3 ≤ 3 ✔️ | 3   | 3        | 停住了，跳不动  |
| 4 | 4       | 4 ≤ 3 ❌  | —   | 3        | 位置 4 到不了 |

所以返回 `false`。

---

# 🎉 小结

贪心法核心思想：

* 只需要看最远能到哪里
* 如果某一步连当前位置都到不了 → 肯定失败
* 一旦最远可达范围覆盖到最后一位 → 肯定成功
