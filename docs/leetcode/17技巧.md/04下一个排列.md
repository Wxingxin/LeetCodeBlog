# 题
整数数组的一个 排列  就是将其所有成员以序列或线性顺序排列。

例如，arr = [1,2,3] ，以下这些都可以视作 arr 的排列：[1,2,3]、[1,3,2]、[3,1,2]、[2,3,1] 。
整数数组的 下一个排列 是指其整数的下一个字典序更大的排列。更正式地，如果数组的所有排列根据其字典顺序从小到大排列在一个容器中，那么数组的 下一个排列 就是在这个有序容器中排在它后面的那个排列。如果不存在下一个更大的排列，那么这个数组必须重排为字典序最小的排列（即，其元素按升序排列）。

例如，arr = [1,2,3] 的下一个排列是 [1,3,2] 。
类似地，arr = [2,3,1] 的下一个排列是 [3,1,2] 。
而 arr = [3,2,1] 的下一个排列是 [1,2,3] ，因为 [3,2,1] 不存在一个字典序更大的排列。
给你一个整数数组 nums ，找出 nums 的下一个排列。

必须 原地 修改，只允许使用额外常数空间。

 

## 示例 1：

输入：`nums = [1,2,3]`
输出：`[1,3,2]`
## 示例 2：

输入：`nums = [3,2,1]`
输出：`[1,2,3]`
## 示例 3：

输入：`nums = [1,1,5]`
输出：`[1,5,1]`

# 题


题干的意思是：找出这个数组排序出的所有数中，刚好比当前数大的那个数

比如当前 nums = [1,2,3]。这个数是123，找出1，2，3这3个数字排序可能的所有数，排序后，比123大的那个数
也就是132

如果当前 nums = [3,2,1]。这就是1，2，3所有排序中最大的那个数，那么就返回1，2，3排序后所有数中最小的那个，也就是1，2，3 -> [1,2,3]

# 解

下面把「下一个排列」（LeetCode 31）用**最稳定的三步法**讲透：**找拐点 → 找交换 → 反转尾巴**。全程 **原地 O(1)**。

---

## 核心结论（记住这三步就能写出来）

给定数组 `nums`：

1. **从右往左找第一个“升序对”**：找到最大下标 `i`，使得
   `nums[i] < nums[i + 1]`
   这个 `i` 叫“拐点/转折点”。

2. **从右往左找第一个比 nums[i] 大的数**：找到下标 `j`，使得
   `nums[j] > nums[i]`
   然后交换 `nums[i]` 和 `nums[j]`。

3. **把 i+1 到末尾反转**（因为原来尾巴是降序，反转后变成最小升序）
   这样得到的就是“刚好比原来大一点”的排列（字典序下一个）。

> 如果第 1 步找不到 `i`（说明整个数组从左到右是非升序/降序），那它已经是最大排列：**直接整体反转成最小排列**。

---

## 案例 1：nums = [1,2,3] → [1,3,2]

### 第 1 步：找拐点 i

从右往左看：

* 看 `2 < 3` ✅，所以 `i = 1`（nums[i]=2）

### 第 2 步：找 j（右侧第一个比 2 大的）

从右往左找：

* `3 > 2` ✅，`j = 2`

交换 i 和 j：

* 交换后：`[1,3,2]`

### 第 3 步：反转 i+1..end

i+1 = 2 到 end=2，只有一个元素，不变
最终：`[1,3,2]`

---

## 案例 2：nums = [3,2,1] → [1,2,3]

### 第 1 步：找拐点 i

从右往左：

* `2 < 1` ❌
* `3 < 2` ❌
  找不到 i，说明整个数组是降序：它是最大排列

直接整体反转：
`[3,2,1]` → `[1,2,3]`

---

## 案例 3：nums = [1,1,5] → [1,5,1]

### 第 1 步：找拐点 i

从右往左：

* `1 < 5` ✅，`i = 1`（nums[i]=1）

### 第 2 步：找 j（右侧第一个比 1 大的）

从右往左：

* `5 > 1` ✅，`j = 2`

交换：
`[1,5,1]`

### 第 3 步：反转 i+1..end

i+1=2..2，不变
最终：`[1,5,1]`

---

## 为什么第 3 步要“反转尾巴”？

因为当你找到拐点 `i` 时，`i` 右侧那一段一定是**从左到右非升序（通常是降序）**。
你在右侧换了一个“刚刚好更大”的数到 `i` 后，为了让整体字典序增长**最小**，右侧必须变成**最小排列**，也就是升序。
而右侧本来是降序，**反转就能变升序**，O(n) 且原地。

---

## JS 代码实现（原地 + 常量空间 + 详细注释）

```js
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var nextPermutation = function(nums) {
  const n = nums.length;

  // 工具函数：交换两个位置
  function swap(a, i, j) {
    const temp = a[i];
    a[i] = a[j];
    a[j] = temp;
  }

  // 工具函数：反转区间 [l, r]
  function reverse(a, l, r) {
    while (l < r) {
      swap(a, l, r);
      l++;
      r--;
    }
  }

  // 1) 从右往左找第一个 nums[i] < nums[i+1] 的位置 i（拐点）
  let i = n - 2;
  while (i >= 0 && nums[i] >= nums[i + 1]) {
    i--;
  }

  // 如果 i < 0，说明整个数组是降序（已是最大排列）
  // 直接反转成最小排列（升序）
  if (i < 0) {
    reverse(nums, 0, n - 1);
    return;
  }

  // 2) 从右往左找第一个 nums[j] > nums[i] 的位置 j
  let j = n - 1;
  while (j >= 0 && nums[j] <= nums[i]) {
    j--;
  }

  // 交换拐点 i 和右侧刚好比它大的 j
  swap(nums, i, j);

  // 3) 反转 i+1 到末尾，使尾巴变成最小（升序）
  reverse(nums, i + 1, n - 1);
};
```
