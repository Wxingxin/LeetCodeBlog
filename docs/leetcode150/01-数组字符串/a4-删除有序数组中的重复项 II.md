# 删除有序数组中的重复项 II
## 题目

给你一个有序数组 `nums` ，请你 原地 删除重复出现的元素，使得出现次数超过两次的元素只出现两次 ，返回删除后数组的新长度。

不要使用额外的数组空间，你必须在 原地 修改输入数组 并在使用 `O(1)` 额外空间的条件下完成。

 

### 说明：

为什么返回数值是整数，但输出的答案是数组呢？

请注意，输入数组是以「引用」方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:

```js
// nums 是以“引用”方式传递的。也就是说，不对实参做任何拷贝
int len = removeDuplicates(nums);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中 该长度范围内 的所有元素。
for (int i = 0; i < len; i++) {
    print(nums[i]);
}
```

### 示例 1：

输入：`nums = [1,1,1,2,2,3]`

输出：`5, nums = [1,1,2,2,3]`

解释：函数应返回新长度 `length = 5`, 并且原数组的前五个元素被修改为 `1, 1, 2, 2, 3`。 

不需要考虑数组中超出新长度后面的元素。
### 示例 2：

输入：`nums = [0,0,1,1,1,1,2,3,3]`

输出：`7, nums = [0,0,1,1,2,3,3]`

解释：函数应返回新长度 `length = 7`, 并且原数组的前七个元素被修改为 `0, 0, 1, 1, 2, 3, 3`。

不需要考虑数组中超出新长度后面的元素。

## Problem Solving

这是 **LeetCode 80：删除有序数组中的重复项 II（中等）**
你这题的关键不是“去重”，而是 **每个元素最多保留 2 个**。

## ✅ 核心思路（一句话先记住）

> **利用有序性，只要保证当前写入的元素，不会让同一个数出现超过 2 次即可**

---

## ✅ 推荐解法：双指针（最常用、最稳）

### 🧠 思想拆解

* `k`：表示 **当前可以写入的位置**（新数组长度）
* 遍历 `nums`
* 前 **两个元素直接保留**
* 从第 3 个开始：

  * 只要 `nums[i] !== nums[k-2]`，就说明不会超过 2 次，可以写

---

## ✨ 代码（带详细注释）

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var removeDuplicates = function(nums) {
  const n = nums.length;
  if (n <= 2) return n; // 少于等于 2 个，肯定全保留

  let k = 2; // 前两个元素一定可以保留

  // 从第三个元素开始遍历
  for (let i = 2; i < n; i++) {
    // 关键判断：
    // 如果当前元素 != 已保留倒数第二个元素
    // 说明这个数最多只出现 2 次
    if (nums[i] !== nums[k - 2]) {
      nums[k] = nums[i];
      k++;
    }
  }

  return k;
};
```

## 🧪 示例 1 详细推演

### 输入

```js
nums = [1,1,1,2,2,3]
```

### 过程

```txt
初始：k = 2
数组：[1,1,1,2,2,3]

i=2 → nums[i]=1
nums[k-2]=nums[0]=1 → 相等 ❌ 跳过

i=3 → nums[i]=2
nums[k-2]=nums[0]=1 → 不等 ✅
nums[2]=2 → k=3

i=4 → nums[i]=2
nums[k-2]=nums[1]=1 → 不等 ✅
nums[3]=2 → k=4

i=5 → nums[i]=3
nums[k-2]=nums[2]=2 → 不等 ✅
nums[4]=3 → k=5
```

### 结果

```js
返回 k = 5
nums = [1,1,2,2,3, ...]
```

## 🧪 示例 2 快速理解

```js
nums = [0,0,1,1,1,1,2,3,3]
```

最终：

```js
[0,0,1,1,2,3,3]
```

因为：

* 每次都只和 `nums[k-2]` 比
* 超过 2 次的第 3、4 个 `1` 被自动跳过

---

## ❗ 为什么是 `k - 2`？

> **因为我们允许最多出现 2 次**

* `k-1`：最后一个保留的
* `k-2`：倒数第二个保留的
  如果当前值 == `nums[k-2]`
  👉 那就是 **第 3 次出现了，必须丢掉**

---

## ⏱️ 复杂度分析（面试必说）

* **时间复杂度**：`O(n)`
* **空间复杂度**：`O(1)`（原地修改）

## 🧠 一句话模板总结（强烈建议背）

> **有序数组去重问题，用快慢指针；
> 保留 k 个，就和 nums[k-保留数] 比**

这句话能直接迁移到：

* LeetCode 26（保留 1 个）
* LeetCode 80（保留 2 个）
* 扩展：保留 `K` 个
