# 反转字符串中的单词

给你一个字符串 `s` ，请你反转字符串中 单词 的顺序。

单词 是由非空格字符组成的字符串。`s` 中使用至少一个空格将字符串中的 单词 分隔开。

返回 单词 顺序颠倒且 单词 之间用单个空格连接的结果字符串。

注意：输入字符串 `s`中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

 

### 示例 1：

输入：`s = "the sky is blue"`

输出：`"blue is sky the"`
### 示例 2：

输入：`s = "  hello world  "`

输出：`"world hello"`

解释：反转后的字符串中不能存在前导空格和尾随空格。
### 示例 3：

输入：`s = "a good   example"`

输出：`"example good a"`

解释：如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。

---

## ✅ 解法一（推荐）：`trim + split + reverse`

### 核心思路

1. 去掉首尾空格
2. 用**正则**按「一个或多个空格」切分
3. 反转数组
4. 用单个空格拼接

---

### ✅ JS 标准代码（面试通过率极高）

```js
/**
 * @param {string} s
 * @return {string}
 */
var reverseWords = function(s) {
  return s
    .trim()                 // 去掉首尾空格
    .split(/\s+/)           // 按一个或多个空格切分
    .reverse()              // 反转单词顺序
    .join(' ');             // 单空格拼接
};
```

---

### 🧠 示例拆解

`s = "  hello   world  "`

* trim → `"hello   world"`
* split → `["hello", "world"]`
* reverse → `["world", "hello"]`
* join → `"world hello"`

---

## ⏱️ 复杂度分析

* **时间复杂度**：`O(n)`
* **空间复杂度**：`O(n)`（新数组）

## 🧩 解法二（进阶）：**从后往前扫描（不使用 split）**

👉 面试官**加分写法**，能体现字符串基本功。

### 思路

* 从右往左扫描
* 跳过空格
* 识别一个完整单词
* 加入结果数组

---

### ✅ JS 进阶代码

```js
var reverseWords = function(s) {
  let res = [];
  let i = s.length - 1;

  while (i >= 0) {
    // 1. 跳过空格
    while (i >= 0 && s[i] === ' ') i--;

    if (i < 0) break;

    // 2. 记录单词结束位置
    let j = i;

    // 3. 找到单词开始
    while (i >= 0 && s[i] !== ' ') i--;

    // 4. 截取单词
    res.push(s.slice(i + 1, j + 1));
  }

  return res.join(' ');
};
```
